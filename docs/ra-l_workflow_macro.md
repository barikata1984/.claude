# RA-L論文執筆ワークフロー（マクロフロー案 v3）

## 設計原則

5層17アイテムは「論文を構成する要素」であり、各アイテムの負担分担欄はその要素を完成させるための作業分担を記述している。本フローは、17の要素をどのような順序と依存関係で完成させていくかを定義する。

アイテムの層番号（意味的深度）と実行順序は一致しない。また、各ステージは「研究上の行為」と「その行為が完成させる論文要素」の両面を持つ。

各ステージの具体的な手順はスキル設計の段階で詰める。本文書はフェーズの順序・依存関係・アイテムとの対応を定義するにとどめる。

## フロー概観

```
Phase A: 研究の方向づけ     Phase B: 技術的実現         Phase C: 論文化            Phase D: 品質保証と投稿
(人間主導)                  (人間主導+エージェント支援)  (エージェント主導+人間承認)  (エージェント主導)

A1 Take-home message        B1 技術アプローチの選定      C1 ストーリーライン設計     D1 クロスモデル査読
A2 Contribution statement   B2 手法の設計・実装・記述    C2 セクション別ドラフト     D2 引用検証・反捏造
A3 研究ポジショニング        B3 実験の実行と結果取得      C3 統合・推敲ループ         D3 テンプレート適用・最終整形
                            B4 図表の作成               C4 ページ予算の調整         D4 投稿前チェックリスト
                            B5 実験の動的再設計
                                → 結果が十分ならPhase Cへ
                                → 実験条件の調整ならB3へ
                                → 技術アプローチ再選定ならB1へ
                                → 研究方向の再検討が必要なら
                                  Phase A再検討を人間に提案

   ├── 関連研究リストと評価 ──────────────────────────────────────┤
   │   (Phase A〜Cにわたり継続的に更新)                          │
   ├── 先行研究との差分の明示 ────────────────────────────────────┤
   │   (Phase A〜Cにわたり継続的に精緻化)                        │
   ├── 比較・アブレーション設計 ──────────────────────────────────┤
   │   (Phase B〜Cにわたり設計→実行→記述)                       │
   └── 査読基準の内部化 ─────────────────────────────────────────┘
       (Phase B〜Dにわたり継続的に適用)
```

### フェーズ完了ゲート

| フェーズ | 完了条件 |
|----------|----------|
| Phase A | 第1層の3アイテム（Take-home message, Contribution statement, 研究ポジショニング）が確定していること |
| Phase B | すべてのcontribution statementに対応する実験結果と図表が揃い、結果がtake-home messageを支持していること。第2層の5アイテムが論文に記述可能な状態にあること |
| Phase C | 全セクションが統合され、ページ制約を満たし、内部的に一貫した論文ドラフトが存在すること |
| Phase D | 引用検証済み、フォーマット準拠、チェックリスト完了の論文PDFが存在すること |

## 横断アイテム

以下の4つのアイテムは特定のフェーズに閉じず、複数フェーズにわたって段階的に完成する。

| アイテム | Phase A | Phase B | Phase C | Phase D |
|----------|---------|---------|---------|---------|
| 関連研究リストと評価 | A3で初期構築（ギャップ探索として） | B1で技術選定の参考として追加検索 | C2でRelated Workとして記述を最終化 | — |
| 先行研究との差分の明示 | A3で骨格を定義 | B3で定量的差分を確定 | C2で対照表・ディスカッションとして記述 | — |
| 比較・アブレーション設計 | — | B1でベースライン候補を選定、B3で実験実行 | C2で設計論理を記述、表・グラフを配置 | — |
| 査読基準の内部化 | — | B3の実験設計時から査読視点を意識 | C3の推敲で自己評価として適用 | D1で外部査読として最終検証 |
